# Excel Business Sales Dashboard

## Project Overview

This project demonstrates how Microsoft Excel can be used to perform business data analysis and build an interactive dashboard. The dashboard analyzes sales data to identify trends, regional performance, and product category contributions.

The goal of this project is to transform raw sales data into meaningful insights using Excel tools such as Pivot Tables, charts, and slicers.

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Visualization

---

## Dataset Structure

The dataset contains the following fields:

| Column   | Description       |
| -------- | ----------------- |
| Date     | Transaction date  |
| Region   | Sales region      |
| Product  | Product name      |
| Category | Product category  |
| Sales    | Revenue generated |
| Profit   | Profit earned     |

---

## Analysis Performed

The following key business questions were analyzed:

* What is the total sales revenue?
* Which region generates the highest sales?
* Which product category contributes the most revenue?
* How do sales change month by month?
* Which categories generate the highest profit?

---

## Dashboard Features

The Excel dashboard includes:

* Total Sales KPI
* Total Profit KPI
* Sales by Region (Bar Chart)
* Sales by Category (Pie Chart)
* Monthly Sales Trend (Line Chart)
* Profit Analysis
* Interactive filters using slicers

Users can filter the dashboard by region and category to dynamically explore the data.
